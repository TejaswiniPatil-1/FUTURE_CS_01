# FUTURE_CS_01 – Vulnerability Assessment and Security Analysis

## Future Interns Cyber Security Internship

### 📌 Project Overview

This repository contains the work completed for **Task 1 – Vulnerability Assessment and Security Analysis** as part of the **Future Interns Cyber Security Internship**.

The objective of this task was to perform a **non-intrusive security assessment** of a publicly accessible web application using industry-standard security tools. The assessment focused on identifying common security weaknesses, analyzing HTTP security headers, and documenting observations with appropriate remediation recommendations.

---

## 🎯 Objective

* Perform a passive vulnerability assessment of a web application.
* Identify common security misconfigurations.
* Analyze HTTP security headers.
* Document findings with risk ratings and recommendations.
* Develop practical skills in security assessment and reporting.

---

## 🛠️ Tools Used

* Nmap
* OWASP ZAP
* Browser Developer Tools
* Git & GitHub

---

## 🔍 Activities Performed

* Network Port Scanning
* Passive Security Analysis
* HTTP Header Inspection
* Security Misconfiguration Review
* Documentation of Findings
* Risk Assessment

---

## 📋 Key Findings

* Missing Content Security Policy (CSP)
* Missing X-Frame-Options Header
* Missing SameSite Cookie Attribute
* Security Header Analysis
* General Security Observations
